# javafmt

For both Java and Kotlin, organize imports and then format the code with either `google-java-format` or `ktlint`.

## Requirements

* `google-java-format`
* `ktlint`

## Example use

```
cd my-java-project
javafmt -w
```

## Quick installation

    go install github.com/xyproto/javafmt@latest
